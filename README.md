每日大赛今日入口
　　刘云志冷笑道：“没关系，我的金刚宝杵出现了一丝神力，我想再过一夜一定可以壮大一些，到时候出其不意，足可以一下子弄死他们两个。”
　　手机中传来李长青发狠的声音，道：“好，既然这样，还有什么可怕的，一定要整死他们！”
　　“王艳，到时候你向柳依依道歉，把她引到一边。我总觉得她的那串念珠不简单，可能也有残存着一点神力。可惜，让你去跟她交换却搞成了这副样子。”刘云志的声音很不满。
　　“好吧！”最终王艳的语气也冷了下来，答应了刘云志的要求。
　　刘云志以低沉嘶哑的声音自语道：“不能在其他人面前动手，这倒是个麻烦，需要好好的谋划一番。”
　　“啪！”
　　庞博合上手机盖，道：“大家都听到了吧，这三个白眼豺狼想整死我与叶凡，这次别怪我们不顾念同学情谊。”
　　这时，叶凡大步向前走去，什么也没有说。庞博紧随其后，张子陵也快步跟了上来，其他人也全部站起，随着他们进入那片树林中。
　　当看到叶凡与庞博大步而来时，李长青与王艳毕竟心里有鬼，顿时有些心惊肉跳，刘云志的面色则阴沉的到极点。
　　“屡屡放过你们，不跟你们计较，而你们却想弄死我们两个，这次再放过你们的话我庞博这两个字倒过来写！”
　　“你……你胡说！”李长青当场就冒出了冷汗。
　　王艳的脸色非常苍白，惊慌地辩解道：“你……你在说什么，我不明白。”
　　庞博也不多说什么，直接再次放了一遍录音。
　　在这一刻，王艳与李长青顿时慌了，而刘云志的脸色也开始发白，在铁一般的事实面前他们根本无法辩解。
　　“我们说的只是气话，这怎么能当真……”李长青大叫，此刻他彻底的恐惧了。
　　“气话？你当在场的同学都是三岁孩子吗？所有人都明白是怎么回事。”向来沉默寡言的张子陵冷冷地开口道。
